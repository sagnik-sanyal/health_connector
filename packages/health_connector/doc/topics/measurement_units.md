# Measurement Units

Type-safe measurement classes with automatic unit conversion.

## Unit Categories

- **Mass**: Kilograms, pounds, grams
- **Length**: Meters, kilometers, miles, feet
- **Energy**: Calories, kilocalories, joules
- **Volume**: Liters, milliliters, fluid ounces
- **Temperature**: Celsius, Fahrenheit
- **Pressure**: Millimeters of mercury (mmHg)
- **Blood Glucose**: Milligrams per deciliter, millimoles per liter

## Features

- Compile-time type safety
- Automatic unit conversion
- Platform-native unit handling
