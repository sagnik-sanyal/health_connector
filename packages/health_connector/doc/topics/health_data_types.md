# Data Types

Supported health and fitness data types across iOS and Android.

## Categories

- **Activity**: Steps, calories, distance, workouts
- **Body**: Weight, height, body fat, BMI
- **Vitals**: Heart rate, blood pressure, blood glucose, SpO2
- **Nutrition**: Macro and micro nutrients
- **Sleep**: Sleep sessions and stages
- **Speed**: Walking and running speed

## Type Safety

Each data type is associated with specific record and measurement types,
providing compile-time type safety for all health data operations.
